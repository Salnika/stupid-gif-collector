# Stupid GIF Collector

Play the 3-attempt GIF catch game, unlock GIFs, and manage your collection.

## Live site

GitHub Pages URL:
https://salnika.github.io/stupid-gif-collector/

## Local development

```bash
yarn install
yarn dev
```

## Build

```bash
yarn build
```

## Deployment

This repository includes a GitHub Actions workflow:
`.github/workflows/deploy-pages.yml`

On every push to `master`, it builds and deploys to GitHub Pages.
